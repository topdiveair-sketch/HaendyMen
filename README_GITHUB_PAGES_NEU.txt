Zuhause am Bach Mobile – GitHub Pages PWA NEU

Wichtig:
- index.html liegt DIREKT im Root dieses ZIPs.
- Für GitHub Pages entweder alle Dateien direkt ins Repository-Root laden
  ODER in einen Ordner mobile/.
- Wenn in Ordner mobile/: Link endet auf /mobile/

GitHub Pages Check:
1. index.html muss im veröffentlichten Pfad liegen.
2. manifest.webmanifest muss erreichbar sein.
3. sw.js muss erreichbar sein.
4. icon.svg muss erreichbar sein.
5. Datei .nojekyll verhindert Jekyll-Probleme.

Chrome:
- Die App muss online per HTTPS laufen.
- Lokale file:// Dateien zeigen oft KEIN „Zum Startbildschirm hinzufügen“.

Funktionen:
- 4 Zimmer
- Gäste speichern/neu hinzufügen
- Buchungen je Zimmer
- Doppelbelegungsschutz
- Extras zur Rechnung
- JSON Export
