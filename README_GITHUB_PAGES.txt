Zuhause am Bach Mobile – GitHub Pages Paket

Installation auf GitHub Pages:
1. ZIP entpacken.
2. Inhalt in dein GitHub-Repository hochladen, z.B. in Ordner mobile/
3. GitHub Pages aktivieren.
4. Link am Android-Handy in Chrome öffnen.
5. Chrome-Menü zeigt dann normalerweise „App installieren“ oder „Zum Startbildschirm hinzufügen“.

Wichtig:
Lokale file:// HTML-Dateien werden von Chrome oft NICHT als installierbare App erkannt.
Über GitHub Pages/HTTPS funktioniert die PWA-Installation deutlich zuverlässiger.

Funktionen:
- 4 Zimmer: DZ Bachblick, Marillenzimmer, Weinbergzimmer, Donauzimmer
- Gäste speichern und neu hinzufügen
- Buchungen je Zimmer
- Doppelbelegungsschutz
- Getränke/Speisen/Frühstück zur Rechnung
- Rechnungstext kopieren
- JSON-Export
